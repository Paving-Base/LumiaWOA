在版本高于 18363 的系统上启用通话

以下是获得并恢复 ICan0 值以使通话在版本高于 18363 的系统中工作的方法。

警告

可能导致设备损坏

本教程会演示正确步骤，但错误仍可能因用户机器上不可预测的情况而发生。我们不对设备的任何数据丢失，软砖或永久性损坏负责。

第一部分 - 获得 ICan0 值

先决条件

  • 一台装有 Windows on ARM build 18363 或更低版本的 Lumia

注意：ICan0 值是每张 SIM 卡特有的，所以你可以用另一部装有目标版本系统的 Lumia 设备来获取。

将 SIM 卡插入系统版本低于 18363 的设备，并开机。

打开搜索，并搜索运行：

regedit

找到以下键值：

HKEY_LOCAL_MACHINE\SOFTWARE\OEM\RILINITSVC\

记下/复制/导出 ICan0 值。这是你在第二部分中需要的，以使通话在任意系统版本中都能正常工作。

尝试以下解决方法：

  • 确保你已经插入 SIM 卡；
  • 重启手机；
  • 确保 MobileNetworkingService 服务正在运行。如果没有启动，尝试将其手动启动并稍等片刻。
  • 你当前的系统版本低于或等于 18363 吗？

第二部分 - 设置 ICan0 值

先决条件

  • 一台装有 Windows on ARM 版本高于 build 18363 的 Lumia

打开搜索，并搜索运行：

regedit

找到以下键值：

HKEY_LOCAL_MACHINE\SOFTWARE\OEM\RILINITSVC\

如果你没有 RILINITSVC 键值，请像截图中一样手动创建一个。

在 RILINITSVC 项中，新建一个字符串值，命名为：

ICan0

并设置为之前你记录下来的值。
