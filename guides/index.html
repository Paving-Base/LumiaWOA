本教程将介绍打开 Lumia 自定义大门的第一步，这是在 Lumia 上安装任何其他受支持的操作系统（如 Windows 10 或 Windows 11）所必需的。

打开教程

安装 Windows

解锁 Bootloader 后，你可以选择以下的两种方法中的一种来在你的 Lumia 上安装 Windows 10/11 On ARM。

通过 WoaDeployer 安装

这个工具可以帮助你使用你所准备的 ISO 镜像轻松安装 Windows On ARM。 此方法可支持双启动。你可以选择 Windows 10/11 ARM64 的任意版本。

打开教程

刷入预先制作好的 FFU 镜像

这是在 Lumia 上安装 Windows On ARM 的一种较为快速但不支持双启动的方法，通过直接刷入预先制作好的 FFU 文件来完成安装。

打开教程

将 Windows on ARM 安装到 SD 卡中

此方法可避免占用内置 EMMC 存储空间，但性能会低于 EMMC。
