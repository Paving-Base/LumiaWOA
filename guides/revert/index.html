还原回 Windows 10 Mobile

玩腻了 WOA，或者想从头开始？我们已为你提供保障。

警告

可能导致设备损坏

本教程会演示正确步骤，但错误仍可能因用户机器上不可预测的情况而发生。我们不对设备的任何数据丢失，软砖或永久性损坏负责。

先决条件

  • Lumia 950 (Talkman) 或 Lumia 950 XL (Cityman)
  • 一台 Windows PC
  • 一条连接手机与电脑的数据线

需要的软件：

  • WPInternals

使用 WPInternals

启动 WPInternals。如果你还没有下载，请从上面的链接中下载，并阅读打开后出现的欢迎指南（看不懂请善用翻译）。

软件打开后会显示如下页面：

前往左侧功能区的"Manual Mode"，并在新打开的页面上点击"interrupt the boot-process"按钮。

将手机连接电脑，WPInternals 会自动识别（某些情况可能需要多试几次），并出现如下的设备信息页面。

前往"Flash"选项卡，并在"Flash original FFU"下选择适合你设备的 FFU 文件，请务必使用微软官方发布的 FFU 文件。

如果没有，请前往“Download"选项卡下并点击"Download All"按钮，其会自动下载你的设备所需的每个文件（包括 FFU）。下载完毕后请返回“Flash“选项卡。

WPinternals 会自动开始刷入，请坐和放宽，如果一切顺利，完成后你会看到如下提示：

同时你的手机会自动启动至你所熟悉的 Windows Phone 系统。
