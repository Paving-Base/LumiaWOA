使用 WoaDeployer 安装 WOA

此教程将描述如何使用 WoaDeployer 在 Lumia 上轻松安装 WOA。

警告

可能导致设备损坏

本教程会演示正确步骤，但错误仍可能因用户机器上不可预测的情况而发生。我们不对设备的任何数据丢失，软砖或永久性损坏负责。

提供的固件仅用于测试目的，我们不对固件引起的任何数据丢失负责。请一定在安装之前备份好你的数据。

手机中的所有数据将被清空！

先决条件

  • 一台 Lumia 950 (Talkman) 或 Lumia 950 XL (Cityman)
  • 一台 Windows 电脑
  • 一根数据线

手机的 Bootloader 必须已解锁。如果你还没有完成解锁，请参考此教程。

Files:

  • 一个从你下载的 ISO 镜像中解压出来的 install.wim 文件
  • WoaDeployer

在开始前...

你需要从比 18363 更旧的版本中提取 ICan0 值。如果你已经拥有，请忽略此框。

如果你没有，请使用相同的指南或者 FFU 刷入教程部署 build 18363 或任何更早的版本然后参考 这个教程提取 ICan0 值，然后再回来部署你想要的系统版本。

在开始之前阅读所有内容！

完成后，不要忘记再次参考此教程恢复 ICan0 值！

进入大容量存储模式

首先，你需要将手机切换到大容量存储模式，以下是 WPInternals 中所需的步骤：

确保关闭任何其他工具，然后打开 WPInternals 并将手机连接电脑。你现在有两个选择：

选项 A) 如果手机已经启动到 Windows Phone，前往 “Manual mode” 选项卡 并选择 “Switch to Mass-Storage-mode”。

选项 B) 如果手机没有启动到 Windows Phone 或者手机没有被检测到，前往 “Manual mode” 选项卡 并点击 “interrupt the boot-process”。然后将手机强制重启：同时按住音量减和电源键，直到手机 关机并振动，WPInternals 应该会识别到你的手机，之后你就可以在 “Manual mode” 选项卡中选择 “Switch to Mass-Storage-mode”。

等到 WPInternals 报告你的手机现在处于大容量存储模式后，关闭 WPInternals。注意：必须关闭，否则接下来的步骤一定会失败。

使用 WoaDeployer

打开 WoaDeployer，你会看到此页面。

认真阅读（看不懂的自己用翻译），并点击“OK”。

这就是 WoaDeployer 的主界面。点击 “Browse...”

选择你的 install.wim 文件。

前往 “Advanced” 选项卡。

在这里选择 Keep Windows 10 Mobile（你可在完成 WOA 安装后启用双启动）。

现在请谨慎选择，因为如果你改变主意，就必须刷回原始 FFU 镜像并从头开始。

完成后，返回 “Deployment” 选项卡。

当你准备好时，按下 "Deploy" 按钮。

程序开始工作，稍等片刻会弹出包含最新更改日志的页面。点击 “Close” 以继续安装。

WoaDeployer 会继续下载需要的文件，并自动开始安装。你可能会看到 Windows 弹出一些有关检测到新设备的通知，忽略即可。

某些时候，系统会提示您再次切换到大容量存储模式，这次需要使用刚刚安装到手机的 Developer Menu（开发人员菜单）。

阅读屏幕上的信息，并按照说明进行操作（长按电源键直到手机黑屏并振动，手机会重启，出现引导菜单时用音量键选择 “Developer Menu”，相机键确认，然后继续用音量键和相机键选择 “UEFI Mass Storage Mode”）。 如果你按照正确的步骤操作，最后你会在手机屏幕上看到 “UEFI Mass Storage” 和 “Connected”。

点击 "Continue"。

在等待这一步的同时，不妨根据自己的喜好来杯咖啡☕或茶🍵。

完成该步骤后，它将注入驱动程序。

...完成！按照显示的步骤操作，尽情享受吧！

提示：如果之前你选择了保留 Windows 10 Mobile 并想使用双启动，不要忘记参考此教程来启用双启动！
