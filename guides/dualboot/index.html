启用与 Windows 10 Mobile 的双启动

在使用 WoaDeployer 安装 Windows On ARM 后，请按照以下教程启用双启动。

Caution

可能导致设备损坏

本教程会演示正确步骤，但错误仍可能因用户机器上不可预测的情况而发生。我们不对设备的任何数据丢失，软砖或永久性损坏负责。

先决条件

  • 一台 Windows 电脑
  • 一台安装好 Windows on ARM 的 Lumia 950 或 950 XL

部署要求

  • 你 必须 使用 WoaDeployer 安装 Windows on ARM。FFU 方式不支持双启动。
  • 你必须在部署前选择 "Keep Windows 10 Mobile" 。如果没有，则必须刷回原厂 FFU （以恢复 Windows 10 Mobile）并重新开始。
  • 将手机启动到桌面（并完成 OOBE 设置）至少一次。

确保在开始之前满足所有要求。

重启手机，等待其振动，然后从引导菜单中选择（使用音量键和相机键来确认）：

Developer Menu -> USB Mass Storage Mode

如果操作正确，手机会显示如图所示的画面：

电脑上打开 WoaDeployer，并将手机连接至电脑。

选择 “Dual Boot” 标签。

点击 "Check Status"。如果你正确地遵循了所有步骤，将出现两个新的按钮来启用或禁用双启动。

点击 "Enable Dual Boot"，稍等片刻，直至弹窗提示完成。然后按住电源键直至手机振动，手机会重启。你现在可以在引导菜单中选择启动 Windows 10 Mobile!
