解锁 Bootloader

本教程将介绍打开 Lumia 自定义大门的第一步。

警告

可能导致设备损坏

本教程会演示正确步骤，但错误仍可能因用户机器上不可预测的情况而发生。我们不对设备的任何数据丢失，软砖或永久性损坏负责。

提供的固件仅用于测试目的，我们不对固件引起的任何数据丢失负责。请一定在安装之前备份好你的数据。

该软件未被批准用于紧急服务。通过安装此软件，您同意可能出现的紧急服务访问中断。请不要将其作为你的主要通讯设备。

先决条件

  • 一台 Lumia 950 (Talkman) 或者 Lumia 950 XL (Cityman)
  • 一台 Windows 电脑
  • 一根数据线

软件：

  • WPInternals

以及充足的耐心！

请在开始操作之前完整阅读教程。 解锁过程涉及许多不同的变量，因此某些步骤可能会有所不同。 可能需要多次尝试。

在开始前...

如果你想保留 Windows 10 Mobile 并与 Windows On ARM 形成双系统，请确保你已经在解锁前将系统更新到最新版本。 解锁 Bootloader 后，你将无法更新 Windows 10 Mobile！

让我们开始吧！

让我们从下载 WPinternals 开始吧。将文件保存到你电脑上的任意位置，并解压。 你可能需要禁用杀毒软件，因为其可能会误报。

运行 WPInternals，你会看到 Getting Started 页面，请仔细阅读，它会告诉你如何安装必要的驱动（通过安装 Windows Device Recovery Tool）。

将手机连接到电脑，WPInternals 应该可以自动识别（如图所示）。 显示信息可能有所不同，无须担心。

你需要一些解锁时所需的文件，包括适用于你的设备的 FFU 系统镜像文件。

如果你没有下载过或者丢失了部分文件，可前往“Download”选项卡并点击"Download All" 按钮，它会为你下载适用于你的设备的所有文件。

准备完毕后，点击左侧的"Unlock Bootloader"。WPInternals 可能会提示你重启到 Flash Mode（如果你的手机在正常模式的话）。 点击“OK”以确定。

你会看到此页面，请花点时间阅读一下。之后，转到页面末尾并点击“Unlock”按钮。 如果按钮无法点击，则需要手动选择正确的 FFU 和 EDE 文件。 如果你还没有下载有关文件，请参照前面的教程下载。

耐心等待，并遵照屏幕上出现的提示操作，然后...

完成！

你的手机现在会正常重启到 Windows Phone。如有需要，你也可以断开与电脑的连接。

接下来的操作？

如果你要安装 WOA，下面有两个方式。

刷入 FFU

下载预先制作好的 FFU 文件并通过 thor2 刷入。此方法较为快捷，但需要有一定的命令行知识，且不支持双启动。

使用 WoaDeployer

使用你自己的 Windows ISO 镜像文件，并让此工具自动完成全部步骤。如果你想保留 Windows 10 Mobile 并形成双启动，必须使用此方法。
