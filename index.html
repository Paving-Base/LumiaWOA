Lumia WOA

适用于 Lumia 的完整 Windows

警告

实验性固件

提供的固件仅用于测试目的。我们不对因我们提供的固件造成的任何数据丢失负责。请在安装之前备份你的数据。

该软件尚未获准用于紧急服务。 安装此软件，即表示你同意不将其用作你的主要电话设备，因为紧急服务访问可能会中断。

你熟悉和喜爱的 Windows

该项目为你的 Lumia 950 和 Lumia 950 XL 带来了桌面版 Windows 10 或 Windows 11 操作系统。

它与你在传统笔记本电脑或台式计算机上使用的 Windows 版本相同，但它是适用于 ARM64 (armv8a) 处理器的版本。

它可以很好的运行 ARM64, ARM, x86 和 x64 应用程序（后两者通过模拟器）。1

融合 Win10 Mobile

MobileShell 是一个功能齐全的自适应 Shell，旨在模仿 Windows 10 Mobile 的外观。

MobileShell 带回了导航栏、状态栏，将通知栏放在顶部，状态图标一目了然，并且仅在你的手机处于平板电脑模式时激活（平板电脑模式仅限 Windows 10）。

MobileShell 还支持横向模式，完美适应手机的当前状态。 MobileShell 由@ADeltaX 制作，默认包含在内！ 你也可以从 Microsoft Store 下载：
