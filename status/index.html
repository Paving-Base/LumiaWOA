需要注意的已知问题

非功能性外设

所有设备上的相机均无法工作，未来也很可能无法工作。

操作系统相关问题

因为 Windows 客户端操作系统的限制，双 SIM 卡不工作。

在双卡设备上，请避免在 SIM 卡插入第二个插槽的情况下启动，这可能会导致问题，即使在移除 SIM 卡后蜂窝网络也可能无法使用。

在版本高于 18363（1909）的系统上，电话功能不会自动注册并启用。你可以参照 此教程来在高版本系统上手动注册。

在 18363（1909）及更低版本上转到蜂窝网络高级选项可能会导致 Windows 设置崩溃。请参阅 Windows Configuration Designer 文档，以便在需要时从外部配置 APN。

短信功能在版本高于 18363（1909）的系统上不可用。

彩信功能在任何 Windows 版本中都不可用。

系统不稳定和性能问题

众所周知，许多非原装或假冒或第三方更换电池会错误地报告正确的容量和充电状态，从而导致间歇性关机、断电和整体系统不稳定。

众所周知，由于 Windows 客户端操作系统比其移动版本资源密集程度更高，因此散热效果与 Windows 10 Mobile 不相上下。
